# About
Welcome to the Metis repository! This project is dedicated to the design, manufacture, and control of a delta robot, entirely from scratch.


# Key Features
- *Custom 3D Printed Components*
- *URDF Description of the robot*
- *ROS2 Integration*
- *Micro-ROS on ESP32*
