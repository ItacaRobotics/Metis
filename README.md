# About
Welcome to the Metis project repository! This project is dedicated to the design, manufacture, and control of a delta robot entirely from scratch.

# Key Features
- **3D Printed Components**
- **URDF Description**
- **ROS2 Integration**
- **Micro-ROS on ESP32**
- **G-Code Parser**
